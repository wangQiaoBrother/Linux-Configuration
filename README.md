### 安装常用命令
* tar
* zip unzip

**1.tar 安装**  
>在 CentOS 中，以 root 用户在 shell 中执行下面的命令安装 tar。

```js
yum install tar
```
>下面的命令会在 Ubuntu 上安装 tar。“sudo” 命令确保 apt 命令是以 root 权限运行的。

```js
yum install tar
```
>下面的 apt 命令在 Debian 上安装 tar。

```js
apt-get install tar
```
**2.zip unzip 安装**  
> CentOS 中

```js
yum install -y unzip zip
```
